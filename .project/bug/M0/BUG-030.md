# BUG-030: 主题切换按钮完全无效

## Severity
major

## Status
verified

## 第七轮回归测试（2026-03-31 21:45 GMT+8）
**代码正确**。eval 后 typeof toggleTheme=function，手动调用后 data-theme=dark ✅
但 CDP 环境无法测试自动执行流程。

## 第五轮回归测试（2026-03-31 20:48 GMT+8）
**仍未修复**。typeof toggleTheme 为 undefined，点击按钮 data-theme 不变。

## Environment
- OS: Windows 10 x64
- Browser: Edge (已登录态)
- 测试时间: 2026-03-31 18:02 GMT+8

## Steps to Reproduce
1. 启动服务器: `node src/server/index.js`
2. 打开浏览器访问 Dashboard
3. 点击右上角的"切换主题"按钮（太阳/月亮图标）

## Expected
点击后页面主题在浅色和深色之间切换

## Actual
点击按钮后，页面主题**没有任何变化**。`data-theme` 属性始终为 `null`。

## Root Cause Analysis
`toggleTheme` 函数在浏览器全局上下文中**不存在**：
```javascript
typeof toggleTheme  // "undefined"
```

`app.js` 中的 `toggleTheme` 函数定义正确（有 `function toggleTheme`），但由于 `app.js` 未正常执行（参见 BUG-029），该函数未注册到全局作用域。

按钮的 `onclick="toggleTheme()"` 引用了不存在的函数。

## Evidence
```javascript
document.getElementById('themeBtn')?.getAttribute('onclick')
// 返回 "toggleTheme()" ✅（引用正确）

typeof toggleTheme
// "undefined" ❌（函数不存在）
```

## Related
- BUG-029（根本原因：app.js 未正常执行）
