## Bug 描述

点击状态筛选按钮（全部/进行中/已完成/阻塞）后，项目卡片未正确过滤。虽然统计栏的数字正确更新，但项目卡片列表仍然显示所有项目。

---

## 修复详情

### 根因
`filterProjects` 函数的筛选条件逻辑错误：

```javascript
// 原代码：if 条件为 false 时继续执行 return true，导致显示所有项目
if (currentFilter === 'in_progress' && (progress === 100 || blocked)) return false;
return true;  // 这行总是执行
```

### 修复方案
重写筛选逻辑为正确的 if-else-if 结构：

```javascript
if (currentFilter === 'done') {
  if (progress < 100) return false;
} else if (currentFilter === 'in_progress') {
  if (progress >= 100 || status === 'blocked') return false;
} else if (currentFilter === 'blocked') {
  if (status !== 'blocked') return false;
}
```

### 修复文件
- `src/public/js/app.js` - `filterProjects()` 函数

---

## 验证结果

✅ **回归测试通过**
- 全部筛选：显示全部 5 个项目
- 进行中筛选：只显示 1 个进行中项目
- 已完成筛选：无符合条件的项目（显示空状态）
- 阻塞筛选：无符合条件的项目（显示空状态）
