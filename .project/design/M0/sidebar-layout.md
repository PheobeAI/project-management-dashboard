## 2. 侧边栏设计

### 2.1 侧边栏规格

| 属性 | 展开状态 | 收缩状态 |
|:-----|:---------|:---------|
| 宽度 | 260px | 60px |
| 高度 | 100vh（固定） | 100vh |
| 背景色 | `#1E293B` → `#0F172A`（渐变） | 同左 |
| Logo 区域 | 220×50px | 40×30px（仅 Logo） |
| 菜单项高度 | 48px | 48px（仅图标居中） |
| 收缩按钮 | 底部，居中 | 变为展开按钮 → |

### 2.2 侧边栏菜单

| 序号 | 栏目名称 | 图标 | 路径 |
|:-----|:---------|:-----|:-----|
| 1 | Dashboard | 🔵 蓝色圆形 | / |
| 2 | Projects | 📁 文件夹 | /projects |
| 3 | Agents | 👥 用户组 | /agents |
| 4 | Tasks | 📋 剪贴板 | /tasks |
| 5 | Versions | 📝 文档 | /versions |
| 6 | Documents | 📄 页面 | /documents |
| 7 | Settings | ⚙️ 齿轮 | /settings |

### 2.3 菜单项样式

**当前选中项**：
```css
.menu-item.active {
  background-color: #334155;
  border-left: 4px solid #60A5FA;
}
```

**悬停状态**：
```css
.menu-item:hover {
  background-color: rgba(255, 255, 255, 0.05);
}
```

**图标与文字间距**：
- 图标：左侧 16px，圆圈 24×24px，居中
- 文字：左侧 12px，font-size: 14px，颜色 `#CBD5E1`

### 2.4 收缩/展开动画

```css
.sidebar {
  transition: width 0.3s cubic-bezier(0.4, 0, 0.2, 1);
}

.sidebar.collapsed {
  width: 60px;
}

.sidebar.collapsed .menu-text,
.sidebar.collapsed .logo-text,
.sidebar.collapsed .collapse-text {
  display: none;
}

.sidebar.collapsed .menu-item {
  justify-content: center;
}
```

## 6. 动画规范

### 6.1 侧边栏收缩动画

```css
.sidebar {
  transition: width 0.3s cubic-bezier(0.4, 0, 0.2, 1);
}

.sidebar.collapsed {
  width: 60px;
}

.sidebar.collapsed .menu-text,
.sidebar.collapsed .logo-text {
  opacity: 0;
  visibility: hidden;
}

.menu-item {
  transition: background-color 0.2s ease, border-color 0.2s ease;
}
```

### 6.2 页面切换动画

```css
.main-content {
  animation: fadeIn 0.3s ease-out;
}

@keyframes fadeIn {
  from { opacity: 0; transform: translateY(10px); }
  to { opacity: 1; transform: translateY(0); }
}
```
