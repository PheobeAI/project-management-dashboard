### 7 个页面设计

| 页面 | 定位 | 核心功能 |
|:-----|:-----|:---------|
| Dashboard | 全局总览，一眼看全局 | 统计概览、项目卡片、阶段分布、测试概览 |
| Projects | 项目详情，深入了解 | 项目信息、Agent状态、任务列表、测试结果、Bug、版本 |
| Agents | 团队工作负载 | Agent列表、任务统计、工作负载可视化 |
| Tasks | 任务流转 | 任务列表、状态筛选、截止时间、优先级 |
| Versions | 版本演进历史 | 版本列表、变更记录、版本对比 |
| Documents | 统一文档中心 | 文档列表、分类、搜索、预览 |
| Bugs | 质量追踪 | Bug统计、趋势图、状态筛选、严重程度 |

### 每个页面的设计要素

每个页面都包含：
1. **页面定位**：解决什么问题
2. **核心功能**：必须有哪些功能
3. **用户流程**：用户怎么用
4. **数据来源**：需要哪些 API/数据
5. **布局结构**：简要的布局描述
