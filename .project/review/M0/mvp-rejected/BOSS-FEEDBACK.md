## 问题清单

### 1. 体验问题
- ❌ **经常有模式对话框弹出"加载项目数据失败"** — 体验很差，经常被打断

### 2. 项目进展显示不合理
- ❌ **项目有多个状态（根据 Software.md），但仪表盘看不出进展到哪一步**
- 需求：根据 Software workflow，项目有多个阶段（Phase 0-8），仪表盘应该清晰展示当前进展

### 3. 功能缺失
- ❌ **没有 Task 进度管理** — 无法查看 Task 状态、进度
- ❌ **没有 Agent 管理** — 无法查看 Agent 状态、todo 列表
- ❌ **没有测试数据管理** — 页面显示"暂无测试数据"
  - 应该显示：不同版本的测试结果、各方评审意见、bug 数量、bug 修复进度

### 4. UI 问题
- ❌ **task 过滤功能有问题** — 过滤后列表为空
- ❌ **设置界面添加项目路径不方便** — 应该调起"选择目录对话框"

### 5. GitHub 版本管理理解错误
- ❌ **当前实现：** Engineer 理解为"在 Dashboard 中管理 GitHub 仓库"
- ✅ **Boss 实际需求：** 这个项目管理系统的源码和进展本身应该提交到 GitHub
  - 现阶段应该提交 GitHub，打 v0.1.0 tag
  - 有阶段性进展就提交
