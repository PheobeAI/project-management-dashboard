## 技术约束

- 状态颜色：
  - waiting: #3B82F6 (蓝)
  - in_progress: #F59E0B (黄)
  - done: #10B981 (绿)
  - cancelled: #64748B (灰)
  - paused: #94A3B8 (浅灰)
- 过期脉冲动画：`animation: pulse 2s infinite`
- 表格使用 `<table>` 或 CSS Grid 实现
