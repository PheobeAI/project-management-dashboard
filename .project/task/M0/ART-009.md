## 验收标准

- [ ] 所有图标 stroke-width 统一（1.5px）
- [ ] 图标颜色可随 CSS 变量变化
- [ ] 入场动画可复用（添加 class 即可）
- [ ] 骨架屏动画流畅（1.5s shimmer）
- [ ] 深色/浅色模式切换无需刷新

---

## 技术约束

- 图标使用 Lucide Icons 风格
- 图标尺寸：16px/20px/24px
- CSS 变量命名：`--color-primary`, `--color-bg`, `--spacing-md` 等
- 动画使用 `will-change` 优化性能
