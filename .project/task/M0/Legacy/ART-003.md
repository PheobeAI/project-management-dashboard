## 验收标准

- [ ] 5 种 Agent 头像可区分（不同颜色/图标）
- [ ] 柱状图动画流畅，延迟错开
- [ ] 悬停效果明显（阴影增强+上浮）
- [ ] 状态分组折叠/展开动画完整
- [ ] 筛选器样式与其他下拉框一致

## 技术约束

- Agent 头像颜色建议：
  - Designer: #8B5CF6 (紫色)
  - PM: #3B82F6 (蓝色)
  - Engineer: #10B981 (绿色)
  - Art: #EC4899 (粉色)
  - QA: #F59E0B (黄色)
- 柱状图使用 Chart.js 或纯 CSS 实现
- 动画使用 transform scaleY 实现
