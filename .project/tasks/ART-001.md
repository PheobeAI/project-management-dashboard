## 验收标准

- [ ] Logo 在深色背景上清晰可见（对比度 ≥ 4.5:1）
- [ ] 项目卡片悬停动画流畅（≥ 30fps）
- [ ] 统计数字递增动画自然
- [ ] 图表颜色与整体风格一致

- [ ] 深色/浅色模式切换正常

## 技术约束

- 使用 SVG 格式确保无限缩放
- 动画使用 CSS transform/opacity 确保性能
- Chart.js 配色需支持主题切换
- 图标使用 Lucide Icons 风格（stroke-width: 1.5px）
