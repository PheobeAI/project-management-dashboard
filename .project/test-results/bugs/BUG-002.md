## Bug 描述

启动 Dashboard 时，端口 3000 被占用导致服务无法启动：

```
Error: listen EADDRINUSE: address already in use :::3000
```

---

## 修复详情

### 根因
`server/index.js` 中端口启动逻辑未处理端口占用情况。

### 修复方案
在 `server/index.js` 中新增端口自动检测逻辑：
- 尝试启动指定端口
- 如果 `EADDRINUSE`，自动尝试下一个端口（+1）
- 最多尝试 10 次

### 修复文件
- `src/server/index.js`
