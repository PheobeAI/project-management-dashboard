## Bug 描述

Dashboard 主内容区右侧半边空白，`.main-content` 未占满剩余宽度。

---

## 修复详情

### 根因
`main.css` 中 `.main-content` 设置了 `max-width: 1400px` 和 `margin: 0 auto`，导致内容居中而非占满宽度。

### 修复方案
1. 移除 `max-width: 1400px` 约束
2. 移除 `margin: 0 auto`（居中）
3. 添加 `width: 100%` 和 `box-sizing: border-box`
4. `sidebar.css` 中 `.main-wrapper` 添加 `flex: 1`

### 修复文件
- `src/public/css/main.css` - `.main-content`
- `src/public/css/sidebar.css` - `.main-wrapper`
