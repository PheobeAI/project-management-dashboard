## 设计概念

**名称**：Project Flow Logo

**核心理念**：用简洁的几何图形表达「项目管理」的核心概念

### 设计元素

| 元素 | 含义 | 颜色 |
|:-----|:-----|:-----|
| 🔵 深蓝色方块 | 项目/任务 | `#3B82F6` → `#1E40AF` 渐变 |
| 🔷 亮蓝色方块 | 活跃任务 | `#60A5FA` |
| ⚪ 浅色连接线 | 工作流程/协作 | `#94A3B8` |
| 🟢 绿色圆点 | 完成状态 | `#10B981` |
| 🟡 黄色圆点 | 进行中状态 | `#F59E0B` |
| ⬛ 深色背景 | 专业沉稳 | `#1E293B` |

### 图形构成

```
    ┌─────────────────────────┐
    │                         │
    │   ■          ■          │  ← 两个深蓝方块代表项目/任务
    │     \        /          │
    │       ■──────■           │  ← 中心亮蓝色方块代表当前活动
    │         │                │
    │    ●    ●    ●          │  ← 三个状态指示点
    │                         │
    └─────────────────────────┘
```

## 使用指南

### 1. 侧边栏 Logo（主 Logo）
```html
<img src="assets/logo/logo-main.svg" alt="Dashboard Logo" width="220" height="50">
```
或使用 CSS 背景。

### 2. Favicon（浏览器 Tab）
```html
<link rel="icon" type="image/svg+xml" href="assets/logo/favicon.svg">
```

### 3. Apple Touch Icon
```html
<link rel="apple-touch-icon" href="assets/logo/apple-touch-icon.png">
```

## 设计备注

1. **SVG 优先**：所有 Logo 均提供 SVG 格式，可无限缩放而保持清晰
2. **深色协调**：主 Logo 设计时考虑了深色侧边栏背景 (`#1E293B`)，对比度足够
3. **简洁原则**：保持图形简洁，确保在 16px 小尺寸时仍可辨识
4. **现代风格**：使用圆角和渐变，符合现代 SaaS 产品审美
